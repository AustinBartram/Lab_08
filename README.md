## Lab Questions

### In plain English describe the function and use of a multiplexer.
A multiplexer is a way for multiiple inputs to gather into a single port and then from there a switch can enable a certain route for those inputs for a desired output. This is a way to reduce wires and to allow for multiple data sources to share a singular channel. In more smple terms the multiplexer allows for multiple connections over a single channel by connecting the output.

### In plain English describe the function and use of a demultiplexer.
A demultiplexer does the opposite of a multiplexer as its job is to take that singular output from a multiplexer and to split it down into the 'original' number of signals(multiple). For more of a definition a demultiplexer can have an input or 2, x1 and x2. And from there it can have multiple outputs like; p1,p2,p3,p4.
